# Calculator App

A sleek, fast calculator built for everyday use.

## What It Does

- Performs basic arithmetic: addition, subtraction, multiplication, division
- Handles decimals and negative values
- Clean interface for fast calculations

## Why Its Cool

- Lightweight and easy to use
- Designed for quick math without distractions
- Reliable results every time

## Usage

1. Open the calculator app.
2. Enter numbers and operations.
3. Hit `=` to see the result.

## Features

- `+` addition
- `-` subtraction
- `*` multiplication
- `/` division
- Clear and reset

## Notes

This calculator app is ideal for simple daily calculations and quick math checks.
